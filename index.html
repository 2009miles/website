About Me

My name is Bruno and I'm a tecnhology enthusiast with a passion for the web.

Over the past few years (and with the encouragement of a few friends) I began tinkering in programming, initially with Python, working my way up from simple command line projects all the way to complete (if not the best looking) websites using frameworks such as Flask, creating simple APIs and managing data through solutions like MongoDB.

Through these experiences I learned two important things:

  • - I really, really enjoyed programming;

  • - I needed to learn frontend development in order to complement the work so far done in the backend or work on anything professionally.

This lead me to find EDIT, the school where over the last few months I improved my knowledge in HTML and CSS (SASS), and where I also had the chance to learn JavaScript, a language which I really enjoy programming in.

Currently I'm working on personal projects, such as this website, and looking for my first job as a Frontend developer where I can start putting my knowledge to work!
